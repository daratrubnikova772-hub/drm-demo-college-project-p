
# DRM и система лицензий на Python

Этот проект — демонстрационная система защиты контента с помощью шифрования и управления лицензиями.  
Позволяет создавать лицензии, проверять их действительность, активировать и использовать для расшифровки файлов.

---

## Основные возможности

- Генерация лицензий с ограничениями по времени и количеству активировок
- Проверка лицензий перед расшифровкой
- Расшифровка и шифрование файлов с использованием симметричного ключа
- Генерация новых ключей для шифрования
- Простое меню управления

---

## Структура проекта

- `crypto_utils.py` — функции для шифрования/расшифровки и работы с ключами
- `license_utils.py` — управление лицензиями (создание, проверка, активация)
- `main.py` — главный скрипт для взаимодействия с пользователем
- `input/` — папка для исходных файлов
- `output/` — папка для зашифрованных и расшифрованных файлов
- `data/keys/` — папка для хранения секретных ключей

---

## Требования

- Python 3.x
- Библиотека `cryptography`

Установка зависимостей:

```bash
pip install cryptography
```

---

## Инструкции по использованию

1. Клонируйте проект или загрузите файлы.
2. Создайте структуру папок (input, output, data/keys).
3. Заполните файл `input/контент.txt` нужным контентом.
4. Запустите `main.py`:
   
```bash
python main.py
```

5. Используйте меню для шифрования, проверки лицензий и дешифровки.
